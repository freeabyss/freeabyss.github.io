XML Schema 简要 01

发表于 2016-12-16 | 分类于 XML |

前言

XML Schema 语言即XML Schema Definition(XSD)，是基于XML格式，用于描述XML结构的文档，也是DTD的替代者。

XML Schema的作用

  • 定义可出现在文档中的元素
  • 定义可出现在文档中的属性
  • 定义哪个元素是子元素
  • 定义子元素的次序
  • 定义子元素的数目
  • 定义元素是否为空，或者是否可包含文本
  • 定义元素和属性的数据类型
  • 定义元素和属性的默认值以及固定值
